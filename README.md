# Avian W chromosome

This repository contains files relevant to the project *Evolutionary history and molecular consequences of avian sex chromosome evolution*

## Contents

1)  **W_chroms.html**: This file contains all the scripts necessary to reproduce the study and to make the figures and tables
2)  **Orthofinder**: This folder contains the results from the Orthofinder study
3)  **Final_alignments**: This folder contains three files, each with the respective multiple species alignments for genes representing ZW chromosomes, Z only chromosomes, and autosomes (chromosome 4)
4)  **Final_trees**: This folder contains the trees generated from each of the ZW genes
5)  **PAML**: This folder contains the results from the PAML runs for genes representing ZW chromosomes, Z only chromosomes, and autosomes (chromosome 4)
